## 说明

index.html  是首页

html 文件夹 存的是静态页面

-   detail 商品详情页
-    info 个人信息页
-    loginandsignup 登陆和注册页
-    productlist 产品列表页
-    shoppingcart 购物车页
-    commonhtml  存的是公用的头部和尾部和渲染模板

js 文件夹存放的是 js业务 待写

libs 文件夹存放的是 要用的第三方库 如jQuery bootstrap require等

scss 文件夹 存放的是 scss文件 所有的css样式

- commonscss 存放的是公共样式

static 文件夹存放的是 img video audio 文件
